# Universal License

## What It Is

The Universal License is an annual subscription that includes all current and future Softorino apps -- Mac and Windows. One license covers both platforms simultaneously (Mac and Windows activate separately).

## What's Included

**Mac apps:** WALTR PRO, SYC PRO, Beamer 4, Folder Colorizer for Mac, PicFindr, CleanAppsNow, DeskMinder

**Windows apps:** AltTunes, WALTR PRO, SYC PRO, iRingg, Folder Colorizer 2, Task ForceQuit Pro 2, Volume Concierge 2, Memory Optimizer 2, Zeus

**Also included:** All future Softorino apps as they are released.

## Pricing

- Annual subscription, auto-renews each year
- Can be cancelled at any time
- Softorino sends reminder emails before renewal
- Subscription managed via: https://softorino.com/submanager/

## Activation

1. Click **"Activate My Apps"** in the top bar after purchasing.
2. Sign in with your email address and follow the instructions sent to your inbox.
3. Download and activate any available app from the License Dashboard.

- One license covers **one macOS device and one Windows device simultaneously**
- You can switch devices anytime via the Subscription Manager: https://softorino.com/submanager/

## Key Point

Individual app purchases are not available. The Universal License is the only way to access Softorino software (except free apps: Zeus, iBetterCharge, WALTR HEIC Converter).

## FAQ

**Is there a free trial?**
Yes. 24-hour free trial. Download the app, launch it, and request a trial key during startup. Note: once the 24-hour trial is activated, it cannot be extended or reset.

**What is the refund policy?**
30-day money-back guarantee. No questions asked. Contact support to request a refund.

**How do I access other apps in my Universal License?**
Download the app from softorino.com, install it, then activate it through your Subscription Manager panel at https://softorino.com/submanager/

**How many devices can I use the license on?**
One macOS device and one Windows device simultaneously. You can switch devices anytime via the Subscription Manager.

**What platforms are supported?**
Depends on the app -- some are Mac only, some Windows only, some both. Check the individual product page or the product list in this file.

# CleanAppsNow

## What It Does

CleanAppsNow fully removes Mac apps including all hidden files, leftover components, orphaned folders, and background tasks that standard deletion misses. Used in support as the recommended way to completely uninstall Softorino apps before reinstalling.

## Key Features

- Complete app removal -- deletes all associated hidden Library files (Application Support, Caches, Cookies, Logs, Preferences, WebKit, etc.)
- Drag-and-drop uninstall
- Categorizes apps as:
  - **Inactive** -- not opened in 6+ months
  - **Leftovers** -- service files remaining after manual removal
  - **Outdated** -- legacy software
- File preview before deletion -- see exactly what will be removed
- Frees up disk space by removing hidden installation files
- Apple-notarized

## Platform

**macOS only.** No Windows version.

## Support Use Case

CleanAppsNow is the recommended uninstall method when troubleshooting Softorino app crashes or persistent issues. It removes files that macOS users cannot access manually (hidden Library folders). Always prefer CleanAppsNow over manual uninstall when asking a customer to do a clean reinstall.

# Legacy Licenses

## What Are Legacy Licenses

Before the Universal License existed, Softorino sold individual app licenses for specific products. Some users purchased these and still have active legacy licenses. They have never migrated to Universal License.

**Legacy apps that were sold individually:**
- Waltr 2
- SYC PRO (older versions, e.g. SYC 2)
- WALTR PRO Lifetime (standalone)

## How to Identify a Legacy License User

Legacy license keys use a **5x5 format**: 5 groups of 5 characters separated by dashes.

Example: `QCE0K-CKG1S-65YPS-IEO0M-TNFTX`

If a user presents a key in this format, they are a **legacy license user -- not a Universal License user.** Do not direct them to the Universal License Dashboard or ask them to activate through it.

## How to Help Legacy License Users

1. Identify which app they purchased (Waltr 2, SYC 2, WALTR PRO Lifetime, etc.) based on what they describe or what they are trying to activate.
2. Direct them to the legacy builds page: https://softorino.com/earlier-versions
3. Ask them to download the correct legacy build for their app and platform.
4. They activate using their 5x5 key directly inside the legacy app -- not through the Universal License Dashboard.

## Important Notes

- A 5x5 key for Waltr 2 will **not** activate WALTR PRO. These are different products.
- Legacy apps may not be fully compatible with the latest macOS or Windows versions -- be honest about this.
- If a legacy user wants access to current apps, they would need to purchase the Universal License.
- Legacy builds download page: https://softorino.com/earlier-versions
